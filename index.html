Introducão

HTML (Hypertext Markup Language) é a linguagem padrão usada para criar páginas web. Ela fornece a estrutura básica de sites, permitindo a organização de conteúdo como texto, imagens, links e outros elementos essenciais. HTML usa "tags" para marcar e definir diferentes partes do conteúdo, sendo fundamental para o desenvolvimento web.

Estrutura Básica de um Documento HTML

Um documento HTML é composto por uma estrutura básica, que inclui as tags essenciais para definir uma página. Abaixo está a estrutura mínima:

<!DOCTYPE html> <html lang="pt-br"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Título da Página </title> </head> <body> <h1>Bem-vindo ao HTML!</h1> <p>Este é um exemplo de um documento HTML básico.</p> </body> </html>
Tags HTML Comuns
  • <html>: Define o início de um documento HTML.
  • <head>: Contém metadados sobre o documento (como título e codificação de caracteres).
  • <title>: Define o título da página que aparece na aba do navegador.
  • <body>: Contém o conteúdo visível da página, como textos e imagens.
  • <h1>,<h2>,<h3>, etc.: Tags para definir títulos, do mais importante (<h1>) ao menos importante (<h6>).
  • <p>: Define um parágrafo.
Tags de Formatação de Texto
HTML oferece várias tags para formatar texto:
  • <b>: Negrito
  • <i>: Itálico
  • <u>: Sublinhado
  • <em>: Ênfase (itálico semântico)
  • <strong>: Importância (negrito semântico)
  • <br>: Quebra de linha
  • <hr>: Linha horizontal
Links e Navegação

Os links são essenciais para a navegação em páginas web. Eles são criados com a tag <a>:

<a href="https://www.exemplo.com">Visite nosso site!</a>

Atributos importantes:

  • href: Define o endereço do link
  • target="_blank": Abre o link em uma nova aba
Imagens

As imagens são adicionadas com a tag <img>. Atributos importantes incluem:

  • src: Define o caminho da imagem.
  • alt: Fornece uma descrição alternativa da imagem (para acessibilidade).
Exemplo: <img src="imagem.jpg" alt="Descrição da Imagem">
Listas

HTML suporta dois tipos principais de listas: ordenadas e não ordenadas.

Lista não ordenada:

<ul> <li>Item 1</li> <li>Item 2</li> </ul>

Lista Ordenada:

<ol> <li>Primeiro</li> <li>Segundo</li> </ol>
Tabelas

Tabelas em HTML são criadas com as tags <table>, <tr>, <td>, e <th>. Exemplo:

<table> <tr> <th>Nome</th> <th>Idade</th> </tr> <tr> <td>João</td> <td>25</td> </tr> </table>
Formulários

Formulários permitem que os usuários enviem dados para um servidor. Eles são definidos pela tag <form>:

<form action="/submit" method="POST"> <label for="nome">Nome:</label> <input type="text" id="nome" name="nome"> <input type="submit" value="Enviar"> </form>
Áudio e Vídeo

HTML5 oferece suporte para incorporar mídia diretamente na página.

Áudio:

<audio controls> <source src="audio.mp3" type="audio/mpeg"> Seu navegador não suporta o elemento de áudio. </audio>

Vídeo

<video controls> <source src="video.mp4" type="video/mp4"> Seu navegador não suporta o elemento de vídeo. </video>
Semântica em HTML

O HTML5 introduziu várias tags semânticas que ajudam na estruturação e na acessibilidade do conteúdo:

  • <header>: Define o cabeçalho da página
  • <footer>: Define o rodapé.
  • <article>: Representa um conteúdo independente.
  • <section>: Define uma seção dentro de um documento.
  • <nav>: Define um bloco de navegação.
Acessibilidade

A acessibilidade web é essencial para garantir que todos, incluindo pessoas com deficiência, possam usar seu site. Algumas práticas incluem:

  • Usar a tag <alt> para imagens.
  • Fornecer uma navegação clara e lógica.
  • Garantir que o conteúdo seja legível por leitores de tela.
Ferramentas de Desenvolvimento
  • DevTools: Ferramentas de desenvolvedor no navegador ajudam a inspecionar e depurar o código HTML.
  • Validador W3C: Ferramenta para validar o código HTML em busca de erros de sintaxe.
